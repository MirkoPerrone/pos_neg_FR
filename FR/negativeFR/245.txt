Je vous signale que le modèle TL-WN725N est décliné en 2 version : v 1 : basé sur le chipset Realtek RTL8188CUS (compatible avec quasiment toutes le version de linux (même raspbian)) v 2 : basé sur le chipset Realtek RTL8188EUS (requiert des compilations manuelles du driver) La version qui m'a été livrée est la version 2. Elle fonctionne mal, après deux jours de compilation des drivers, sur mon Raspberry Pi (raspbian linux). Je vous déconseille de l'acheter si vous voulez l'utiliser avec un linux.
