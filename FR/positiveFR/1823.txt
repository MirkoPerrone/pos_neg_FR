Câble d'excellente qualité. Le fil ne craint ni d'être déformé ni de se casser, et la connexion avec les slots USB et microSD se fait très bien, sans aucun jeu, le maintien est parfait. Par contre, certains commentaires parlent d'un taux de transfert accéléré par rapport à d'autres câbles. Chronomètre à l'appui, voilà ce que ça donne, en comparaison avec le câble fournit dans le package initiale de mon HTC M8, pour un transfert d'un même dossier de 300Mo : Transfert sur le téléphone : - Avec le câble de base : ~23-24 Mo/s - Avec le câble Anker : ~25Mo/s Transfert sur la carte microSD : - Avec le câble de base : ~13 Mo/s - Avec le câble Anker : ~14-15Mo/s Bref, le câble Anker permet bien d'accélérer le temps de transfert, mais le gain est vraiment négligeable (en tout cas, sur mon téléphone, après sur d'autres je ne peux pas dire).
