"Ce mini-clavier vous permet de gérer votre ordinateur media center. Le packaging: Le clavier est emballé dans une petite boite cartonnée, elle contient aussi une notice en anglais et un câble micro-usb pour la recharge qui possède lui-même une entrée USB pour une clé USB par exemple. Ça évite de se priver d'un port. Le produit: Le clavier est très compact, il mesure 26cm x 8,3cm x 13,5cm pour 210g. Il possède un interrupteur on/off sur la tranche et le port micro usb. Le récepteur RF est inclus d'un emplacement prévu au sein du clavier, très pratique. L'ensemble est assez réussi et la finition de très bonne qualité. Sur la face du clavier, une led qui informe de la charge et de la réception, le touchpad et les boutons dédiés à la souris et le reste des touches d'un clavier classique (format AZERTY). Une touche ""Fn"" (fonction) comme sur les pc portables permet quelques raccourcis supplémentaires : ouvrir un navigateur, la message, le logiciel media center. Il possède un petit pied qui permet de surélever le clavier, qui lui permet d'être destiné à de la bureautique sur un support stable. Son principale atout est le touchpad intégré qui permet de simuler la souris depuis le clavier directement. A l'usage: Avec Windows 8.1 le clavier est automatiquement détecté. D'après la notice le clavier semble être compatible Mac, Linux et PC. La batterie est un bon point, ça permet de se passer de pile, l'autonomie n'est pas indiquée sur la notice. La prise en main ainsi que la frappe sont excellentes, toute les touches sont présentes ce qui n'est pas le cas de tous les mini-clavier media center (je n'ai la touche ""\"" sur ma Logitech Di Novo mini par exemple). Le mode ""Air mouse"" est vraiment appréciable sur XBMC par exemple, c'est réellement le gros atout de ce clavier. Pour modifier les applications lancées par défaut, il suffit de modifier les applications par défaut de Windows : il suffit de rechercher ""programmes par défaut"" dans le menu démarrer et de modifier les logiciels par défauts. J'ouvre XBMC/Kodi à la place de Windows media player. Attention, ce clavier n'est pas Bluetooth, mais ""radiofréquence"" avec un dongle USB. Il semble être compatible avec les tablettes équipées d'un port USB et possède un récepteur USB ""OTG"". En conclusion, Les +: - Excellente finition - Très compact - Touchpad - Le câble USB avec un port USB - Le prix Les -: - RAS Je recommande ce clavier, pour un très bon rapport prestations/qualité/prix."
